## Resumo do jogo:
<p align= "justify">
O jogo Emotiquiz trás uma dinâmica descontraída para o terminal, misturando a mecânica de dois jogos: quiz e wordle. Nele os jogadores precisam adivinhar, através de dicas, qual é o filme correto e representá-lo por 5 emojis.
</p>


| Plataforma | Publico alvo | Modo do jogo | Gênero  |
|:---------:|:------------:|:----------:|:----------:|
|Terminal MacOs | Usuários de MacOs com interesse em jogos de terminal | Single Player | Casual |
